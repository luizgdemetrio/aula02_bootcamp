# aula02_bootcamp

1. Inteiros (int)

    Métodos e operações:
        + (adição)
        - (subtração)
        * (multiplicação)
        // (divisão inteira)
        % (módulo - resto da divisão)

2. Números de Ponto Flutuante (float)

    Métodos e operações:
        + (adição)
        - (subtração)
        * (multiplicação)
        / (divisão)
        ** (potenciação)

3. Strings (str)

    Métodos e operações:
        .upper() (converte para maiúsculas)
        .lower() (converte para minúsculas)
        .strip() (remove espaços em branco no início e no final)
        .split(sep) (divide a string em uma lista, utilizando sep como delimitador)
        + (concatenação de strings)

4. Booleanos (bool)

    Operações lógicas:
        and (E lógico)
        or (OU lógico)
        not (NÃO lógico)
        == (igualdade)
        != (diferença)
